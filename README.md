# ✅ To-Do List App

A simple and clean To-Do List web application built with plain **HTML**, **CSS**, and **JavaScript** — no frameworks, no libraries, just pure beginner-friendly code.

---

## 🖥️ Live Preview

> Open `index.html` in your browser to run the app locally.

---

## ✨ Features

- Add new tasks by typing and pressing **Enter** or clicking **+ Add**
- Mark tasks as **done** with a checkbox
- **Delete** individual tasks
- Filter tasks by **All / Active / Done**
- Shows count of remaining tasks
- **Clear completed** tasks in one click

## 🚀 How to Run

1. **Clone the repository**
   ```bash

   ```

2. **Open the project folder**
   ```bash
   cd todo-list
   ```

3. **Open in browser**
   - Double-click `index.html`, OR
   - Right-click → *Open with* → your browser

No installation or setup needed. ✅
